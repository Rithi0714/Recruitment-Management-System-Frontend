# Recruitment Management System (RMS)

## Overview

The Recruitment Management System (RMS) is a modern web application designed to streamline and manage the recruitment process. The platform provides dedicated dashboards for Admins, HR personnel, and Recruiters, enabling efficient candidate tracking, interview management, recruiter coordination, and hiring workflows.

## Features

### Admin Dashboard

- View recruitment statistics and insights
- Manage candidates and recruiters
- Track interview progress
- Monitor recruitment activities

### HR Dashboard

- Manage candidate records
- Schedule and monitor interviews
- Track selected candidates
- Manage recruitment settings

### Recruiter Dashboard

- Manage assigned candidates
- Track interview status
- Update candidate progress
- Monitor recruitment activities

### Candidate Management

- Add and manage candidate profiles
- Track candidate status
- Maintain recruitment records

### Interview Management

- Schedule interviews
- Track interview outcomes
- Manage interview workflows

### Activity Tracking

- Record recruitment activities
- Maintain activity logs
- Improve transparency and accountability

## Installation

### Clone the Repository

```bash
git clone https://github.com/Anjali112005/Recruitment-Management-System.git
cd Recruitment-Management-System
```

### Install Dependencies

```bash
npm install
```

### Run the Application

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
